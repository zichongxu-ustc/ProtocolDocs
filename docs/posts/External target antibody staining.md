# 内标抗体染色
<!-- more -->
## 试剂
- eFlour780
- 外标抗体
- FACS Staing Buffer
- PBS
- Beads?



## 细胞处理 

细胞加入PBS重悬，300-600g 5min，弃上清

## 死活染色
用PBS将eFlour780 1：10000稀释（0.1μL->1mL），96孔板每孔50μL、EP管每管100μL（细胞量少时酌情减少体积），RT 8min，加入一倍体积FACS Staing Buffer终止，300-600g 5min，弃上清

## 外标染色
使用FACS Staing Buffer稀释抗体，每孔/管 50μL 吹匀，4℃ 30min（难染抗体37℃ 30min，分开染），加入1-4倍体积FACS Staing Buffer，300-600g 5min，弃上清

200μL FACS Staing Buffer重悬上机

## 单标染色

将震荡后的Beads挤出完整一滴，按50μL/抗体加入FACS Staing Buffer，分50μL/管，加入同比例抗体，RT 15min，加入1-4倍体积FACS Staing Buffer，300-600g 5min，弃上清

200μL FACS Staing Buffer重悬上机
