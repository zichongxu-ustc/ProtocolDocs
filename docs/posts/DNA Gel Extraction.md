# 核酸胶回收
<!-- more -->
## 试剂
- SPARKeasy Gel/PCR Purification Kit (目录号：AE0101)
## 操作

1. 柱平衡：硅胶膜吸附柱放入收集管，向吸附柱硅胶膜中间部位加入100μL的平衡液，12000rpm 离心1min，倒掉收集管中废液，将吸附柱重新放回收集管中，备用。
2. 在长波紫外灯下，用干净刀片将所需回收的DNA条带切下，放入1.5mL离心管，称重。
3. 加1μL/mg 溶胶/结合液DB，放置于56℃水浴锅中溶胶，直至胶完全溶解（5-10min），期间每2-3min上下颠倒几次帮助加速溶解。
4. 溶液平衡至室温后，瞬离，加入吸附柱中，室温放置1min，12,000 rpm离心30-60 s，弃去收集管中废液。
5. 加入600μL漂洗液WB(注意加乙醇)
6. 将吸附柱放回空收集管中，12,000rpm 30-60s，弃去收集管中废液。
7. 吸附柱放回空收集管中，12,000rpm 2min，弃去收集管
8. 取出吸附柱EC，放入一个干净的离心管中，在吸附膜的中间部位加30-50μL 65-70℃洗脱缓冲液EB，12,000rpm 1min。
9. 第6步及之前所有离心步骤可用泵代替
